# Project Overview
This project demonstrates hosting a static website (HTML, CSS, JS) for free using GitHub Pages.It can serve as a personal portfolio, demo site, or any small website project.

# Features
• Fully static website (HTML, CSS, JS)
• Mobile-responsive design
• Hosted for free on GitHub Pages
• Easy to update by pushing commits
